# Pippin

**Pippin** is a highly polished, customisable client for [Plex](https://www.plex.tv/) and [Jellyfin](https://jellyfin.org/) — with a built-in player that rivals Infuse. Available on iPhone, iPad, Mac, and Apple TV.

This repository is the public home for **bug reports, feature requests, and discussion**. The app's source code lives in a separate private repository.

## Found a bug or have an idea?

- 🐛 [Report a bug](https://github.com/lmor152/pippin-player/issues/new?template=bug_report.yml)
- 💡 [Request a feature](https://github.com/lmor152/pippin-player/issues/new?template=feature_request.yml)
- 💬 [Ask a question or start a discussion](https://github.com/lmor152/pippin-player/discussions)

When reporting a bug, please include your platform and version — the issue form will ask for these. If you open the form from inside Pippin (**Settings → Report an issue**), your platform and version details are filled in automatically.

## Tips before filing

- Search [existing issues](https://github.com/lmor152/pippin-player/issues) first — someone may have already reported it.
- For bugs, exporting your logs (**Settings → Logs**) and attaching them helps a lot.
